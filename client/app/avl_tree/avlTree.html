This visualization demonstrates how elements are added to an AVL binary search tree. The AVL tree was the first self-balancing binary search tree to be invented, in 1962! It operates under the invariant that given any node, its left and right subtrees can have a height difference no greater than one. If a node is inserted that violates this property, the tree auto-rotates to maintain both the BST ordering and the height invariant.
